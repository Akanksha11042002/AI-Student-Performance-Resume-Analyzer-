# AI Student Performance & Resume Analyzer (Python)

## Project Overview  

This project analyzes a student's resume to identify skills, detect missing skills, and evaluate overall performance.  
It also includes a typing speed test and study planner to help improve learning efficiency.

## Core Modules  

•	Resume Analyzer  
•	Typing Speed Test  
•	Study Planner  
•	Performance Report  

## Key Analysis  

•	Skill Detection from Resume  
•	Missing Skills Identification  
•	Performance Score Calculation  
•	Typing Speed Measurement (WPM)  

## Tools Used  

•	Jupyter Notebook

## Key Insights  

•	Students can identify their skill gaps easily  
•	Typing speed helps evaluate efficiency  
•	Study planner provides structured improvement plan  
•	Higher skill match leads to better performance score
